# Vote Dapp

## Project setup
```
npm install
npm install ganache -g
npm install truffle -g
```

### Start HttpProvider
```
ganache
```

### Deployed contract
```
truffle migrate
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
